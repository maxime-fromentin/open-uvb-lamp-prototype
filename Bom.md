# BOM Notes

This file is an informal sourcing note. The structured public BOM is `digikey_bom_clean.csv`, and the KiCad-oriented relay BOM is `kicad_uvb_project/BOM_mechanical_relay.csv`.

## Main Items

- G23 lamp holder.
- Philips PL-S 9W/01/2P narrowband UVB lamp.
- Electromagnetic ballast compatible with the selected lamp.
- EN 170 or equivalent UV protective goggles.
- UV protective sleeve or shielding for non-target skin.
- AC fan or DC fan, depending on final enclosure design.
- AC-to-5V power module for the low-voltage control electronics.
- Seeed XIAO RP2040 relay controller.
- External mains cord and appliance inlet.
- Main AC switch.
- Fuse holder and fuse on live conductor.
- Polished aluminum reflector.
- Perfboard or prototyping board for the control circuit.
- Headers or sockets for removable controller mounting.
